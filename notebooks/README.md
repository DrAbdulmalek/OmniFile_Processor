# 📓 Notebooks — دليل الاستخدام

هذا المجلد يحتوي على جميع الدفاتر التفاعلية (Jupyter Notebooks) الخاصة بمشروع OmniFile AI Processor.

---

## 📋 قائمة الدفاتر

| الملف | الغرض | الحالة |
|-------|-------|--------|
| `OmniFile_Diagnostic.ipynb` | فحص صحة جميع الوحدات والتشخيص (25 خلية) | ✅ حديث (v4.1) |
| `OmniFile_Gradio_Debugger.ipynb` | واجهة تفاعلية للاختبار والتصحيح عبر Gradio | ✅ حديث (v4.1) |
| `OmniFile_Complete_v4.ipynb` | سير العمل الكامل — الإصدار v4 | ✅ حديث (v4.x) |
| `OmniFile_Complete.ipynb` | سير العمل الكامل — نسخة v3 | 📦 أرشيف |
| `HandwrittenOCR_Colab.ipynb` | مشروع HandwrittenOCR الأصلي (Google Colab) | 📦 أرشيف |
| `HandwrittenOCR_Ultimate.ipynb` | نسخة موسعة من مشروع HandwrittenOCR | 📦 أرشيف |
| `OmniFile_v3_Testing.ipynb` | اختبارات الإصدار v3 | 📦 أرشيف |

---

## 🚀 للبدء السريع

1. **أفضل نقطة بداية:** افتح `OmniFile_Diagnostic.ipynb` — يفحص جميع الوحدات ويُظهر حالة النظام
2. **للاختبار التفاعلي:** استخدم `OmniFile_Gradio_Debugger.ipynb` — واجهة كاملة للمعالجة والبحث
3. **للتشغيل على Colab:** أي دفتر يعمل مباشرة على Google Colab مع `!pip install -r requirements-colab.txt`
