---

## نبذة عن مشروع OmniFile_Processor

مشروع متكامل للتعرف على الخط العربي اليدوي (OCR) مع تصحيح إملائي ذكي وبوابة ذكاء اصطناعي متعددة النماذج.

- **241** ملف Python | **65,505** سطر كود | **336** اختبار
- **3** محركات OCR: TrOCR + EasyOCR + Tesseract
- **HybridSpellChecker v7.0** مع قاموس CorrectionRule
- **AI Gateway** بـ 80 ملف في `modules/ai/gateway/`
- واجهة Gradio على HF Spaces بـ **8** تبويبات
