# OmniFile AI Processor

نظام ذكاء اصطناعي متكامل يجمع بين ثلاث وحدات أساسية:

1. **وحدة المعالجة النصية (NLP):** تصنيف النصوص، استخراج الكيانات، الترجمة التقنية، التصحيح الإملائي
2. **وحدة الرؤية الحاسوبية (CV & OCR):** استخراج النصوص من PDF والصور (TrOCR + EasyOCR + Tesseract)
3. **وحدة التنظيم والحماية:** فرز الملفات، حماية الأكواد، التعامل مع الأرشيفات المحمية

## Features

- 🔤 Dual-language spell correction (Arabic + English)
- 📄 PDF processing with PyMuPDF
- 🔍 Text classification into 6 categories
- 🌐 English to Arabic technical translation
- 🏷️ Named Entity Recognition (NER)
- 🔐 Code protection during processing
- 📁 Automatic file organization
- 🔎 Security scanning for sensitive data
