# Mobile Review — دليل الاستخدام

## المجلدان ومتى تستخدم كل منهما

### `mobile/` — PWA ثابت للمراجعة المحلية
- **ملف واحد**: `ocr-review.html` يعمل مباشرة في المتصفح
- **لا يحتاج خادم**: افتح الملف مباشرة أو عبر `file://`
- **يعمل بدون إنترنت**: بعد التحميل الأول (Service Worker)
- **الاستخدام**: مراجعة سريعة شخصية على جهاز واحد
- **الحدود**: لا يمكن حفظ التصحيحات على السيرفر، لا تعاون جماعي

### `mobile_review/` — خادم Flask كامل للمراجعة عن بُعد
- **REST API**: نقاط نهاية للتحميل والحفظ (`/save`, `/load`)
- **يعمل على الشبكة**: يمكن الوصول من أي جهاز على نفس الشبكة
- **دعم ngrok**: يمكن الوصول من الإنترنت
- **الاستخدام**: مراجعة جماعية، فريق عمل، مراجعة من أجهزة متعددة
- **الميزات**: حفظ تلقائي، استئناف من جلسة سابقة، دعم الجداول

### متى تستخدم أيهما؟

| الحالة | الأفضل |
|--------|--------|
| مراجعة سريعة على هاتفي | `mobile/` |
| أريد تعديل النص ونسخه فقط | `mobile/` |
| فريق يراجع نفس المستند | `mobile_review/` |
| أريد حفظ التصحيحات تلقائياً | `mobile_review/` |
| العمل من Colab أو جهاز آخر | `mobile_review/` |
| لا يوجد خادم متاح | `mobile/` |

### تشغيل mobile_review/

```bash
# تثبيت Flask (مرة واحدة)
pip install flask

# تشغيل الخادم
python mobile_review/server.py --port 5000

# من الهاتف على نفس الشبكة
# http://192.168.x.x:5000

# من الإنترنت عبر ngrok
ngrok http 5000
```
